Conceito
Identidade visual
Pense nas "configurações permanentes" da marca — cores, tipografia, forma. Ela define como a marca aparece para o mundo.
Arquivo
Brand guide
É o documento que guarda todas as regras. Vive dentro do projeto, mas orienta tudo — da embalagem ao post do Instagram.
Ferramenta
Token de design
É como uma "variável de cor ou espaço" que o projeto usa como referência. Cada decisão visual vira um token reutilizável.
